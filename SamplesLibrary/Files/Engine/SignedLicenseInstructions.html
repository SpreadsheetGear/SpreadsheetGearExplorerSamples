SpreadsheetGear Signed License

To avoid running into exceptions relating to the "Free" mode of SpreadsheetGear for Windows, you will need to activate either the 30-Day Trial mode or fully Licensed mode of the product by calling SpreadsheetGear.Factory.SetSignedLicense(...) with your Signed License string in MainWindow.xaml.cs.

If you are a licensed user of SpreadsheetGear, generate a Signed License from the SpreadsheetGear License User Downloads page at:
https://www.spreadsheetgear.com/Downloads/Licensed

If you are evaluating SpreadsheetGear, please visit the following page to generate a Trial Signed License string:
https://www.spreadsheetgear.com/Downloads/Evaluation

The capabilities of the Trial mode are identical to the Licensed mode but is limited to 30 days.
